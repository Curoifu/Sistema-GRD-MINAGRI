Bienvenido al Sistema GRD

Una plataforma para la gestión y monitoreo de amenazas agroclimáticas en el sector agrícola.

🏚️ ¿Qué es el Sistema GRD?

El Sistema de Gestión del Riesgo de Desastres del MINAGRI permite visualizar, documentar y actuar frente a amenazas que afectan al sector agrícola, promoviendo una respuesta oportuna y eficiente.
